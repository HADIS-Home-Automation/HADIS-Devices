# HADIS - Wi-Fi Switch Plug version

MQTT relay switch wall plug version.
Start setup mode by pressing the button for approximately 6sec.

## Electronics
* Device: [Sonoff S20](https://sonoff.tech/product/wifi-smart-plugs/s20)

## Pins
* Button: GPIO 0
* Relay: GPIO 12
* Green LED: GPIO 13
